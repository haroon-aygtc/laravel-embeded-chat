Laravel Embedded Chat Widget

This documentation helps you integrate the chat widget into any website. The widget is designed to be lightweight, customizable, and easy to implement.

Quick Start

Add the following code to your website to integrate the chat widget: 

<script src="https://your-domain.com/embed.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    LaravelChat.init({
      widgetId: 'your-widget-id',
      position: 'bottom-right',
      primaryColor: '#4f46e5'
    });
  });
</script>
Tip: Replace your-widget-id with the ID of your chat widget from the admin dashboard.

Integration Methods

There are two ways to integrate the chat widget into your website:

1. Floating Widget

The floating widget appears as a chat button in the corner of your website. When clicked, it expands into a chat window. 

<script src="https://your-domain.com/embed.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    LaravelChat.init({
      widgetId: 'your-widget-id'
    });
  });
</script>

2. Embedded Widget

The embedded widget is placed within a specific container on your website. 

<script src="https://your-domain.com/embed.js"></script>
<div id="chat-container" style="width: 100%; height: 600px;"></div>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    LaravelChat.init({
      widgetId: 'your-widget-id',
      container: 'chat-container'
    });
  });
</script>

Configuration Options

The chat widget supports the following configuration options: 

{
  // Required
  widgetId: 'your-widget-id',  // The ID of your widget from the admin dashboard
  
  // Optional
  container: null,              // ID of the container for embedded mode
  position: 'bottom-right',     // bottom-right | bottom-left | top-right | top-left
  primaryColor: '#4f46e5',      // Primary color in hex format
  autoOpen: false,              // Automatically open the widget when the page loads
  buttonSize: '60px',           // Size of the chat button
  widgetWidth: '380px',         // Width of the chat window
  widgetHeight: '600px',        // Height of the chat window
}

Customization Examples

Example 1: Custom Colors and Position

LaravelChat.init({
  widgetId: 'your-widget-id',
  position: 'bottom-left',
  primaryColor: '#10b981'  // Green color
});

Example 2: Auto-Open and Custom Size

LaravelChat.init({
  widgetId: 'your-widget-id',
  autoOpen: true,
  buttonSize: '70px',
  widgetWidth: '420px',
  widgetHeight: '700px'
});

Example 3: Embedded in a Specific Container

<div id="support-chat" style="width: 100%; height: 500px; border: 1px solid #e5e7eb; border-radius: 8px;"></div>

<script>
  LaravelChat.init({
    widgetId: 'your-widget-id',
    container: 'support-chat'
  });
</script>

API Reference

Methods

The chat widget exposes the following methods: 

// Initialize the widget
LaravelChat.init(options);

// Manually open the chat widget
LaravelChat.open();

// Manually close the chat widget
LaravelChat.close();

// Destroy the widget (remove it from the page)
LaravelChat.destroy();

Best Practices

  • Place the widget script at the end of your page or use the DOMContentLoaded event to ensure all DOM elements are loaded before initializing.
  • Use a color that matches your website's branding for the primary color.
  • Consider using the auto-open feature for support pages where users are actively seeking assistance.
  • For mobile sites, you may want to use the bottom-right position as it's easier for thumb access.
Important: The chat widget loads resources from your server. Ensure your server can handle the additional traffic and that the widget URL is correctly configured.

Troubleshooting

If you encounter issues with the chat widget, check the following:

  • Ensure you have the correct widget ID.
  • Check browser console for any JavaScript errors.
  • Verify that your website's domain is allowed in the widget settings.
  • If using CORS restrictions, ensure that your domain is properly configured in the widget settings.
