Laravel Embedded Chat Widget Demo

Floating Chat Widget Example

This example shows the chat widget as a floating button in the corner of the page.

<script src="embed.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    LaravelChat.init({
      widgetId: 'your-widget-id',
      position: 'bottom-right',
      primaryColor: '#4f46e5'
    });
  });
</script>

The widget should appear in the bottom-right corner of this page.

Embedded Chat Widget Example

This example shows the chat widget embedded directly within a container on the page.

<div id="embedded-chat" style="width: 100%; height: 500px;"></div>

<script>
  LaravelChat.init({
    widgetId: 'your-widget-id',
    container: 'embedded-chat'
  });
</script>

Chat Controls Example

You can programmatically control the chat widget:

// Open the chat widget
LaravelChat.open();

// Close the chat widget
LaravelChat.close();

// Remove the chat widget from the page
LaravelChat.destroy();
